## Install

```vim
call dein#add('utubo/vim-tablist')
nnoremap <Leader>T <Cmd>call tablist#Show()<CR>
```

## Keymaps
```
q Close Tablist
<CR> Show tab
r Refresh list
d Close tab
D Close tab (fource)
J Move to right
K Move to left
o New tab (right)
O New tab (left)
```
